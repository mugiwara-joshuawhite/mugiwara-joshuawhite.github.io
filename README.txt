Monetally is a financial support website where the user can get a broader view of their finances, and help themselves be more financially literate.
A user may:
- Input income streams and regular expenses.
- Set aside a certain percentage of income.
- See how their past month has been going.
- Get notified of upcoming payments.
- Read articles that assist them in becoming financially literate.

This website is meant for those who are looking to improve their financial literacy.
By using this website, a user gets a broader view of their finances, which may make them easier to manage.

To get started with Monetally, a user can visit the GitHub page https://mugiwara-joshuawhite.github.io/.
The first page involves a simple sign-in screen. The user account is saved securely in the browser storage.
Upon signing up, a user will br prompted to add income streams, expenses, and distributions. 
These initial streams and expenses are assumed to go forever. To edit, add, or remove streams or expenses, there are income and expenses pages.
